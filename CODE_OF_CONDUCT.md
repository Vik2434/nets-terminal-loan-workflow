# Code of Conduct

## Our Standards

We want this repository to remain welcoming, practical, and respectful.

Examples of positive behavior:

- being respectful in reviews and discussions
- focusing feedback on the code and documentation
- sharing context clearly and constructively
- helping future maintainers understand tradeoffs

Examples of unacceptable behavior:

- harassment or personal attacks
- hostile or dismissive review comments
- publishing sensitive internal data
- intentionally disruptive behavior

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

## Scope

This code of conduct applies to:

- issues
- pull requests
- discussions
- documentation contributions
