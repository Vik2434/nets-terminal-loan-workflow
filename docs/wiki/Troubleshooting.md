# Troubleshooting

## Common Issues

- Access denied on protected pages
- Placeholder values still present in Config
- Email links pointing to the wrong deployment
- Empty tech or finance queues
- Request not found in tracking

## First Checks

1. Confirm the deployed web app URL
2. Confirm the spreadsheet ID
3. Confirm the signed-in user domain
4. Confirm `User_Roles`
5. Confirm `Config` recipients and page URLs
