# Configuration

## Main Configuration Points

- `Config.gs` for bootstrap values
- `Config` sheet for runtime values
- `User_Roles` sheet for access control

## Must Replace

- spreadsheet ID
- deployed web app URL
- allowed domain
- role users
- notification recipients
