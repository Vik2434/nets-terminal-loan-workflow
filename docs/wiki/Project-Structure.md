# Project Structure

## Core Files

- `Code.gs`: routing and boot payload
- `Config.gs`: project constants and config readers
- `Data.gs`: sheet access, normalization, and access parsing
- `Setup.gs`: sheet/config/role setup
- `request.gs`: request submission
- `tracking.gs`: tracking and reminders
- `admin.gs`: tech dashboard
- `approval.gs`: approval dashboard
- `finance.gs`: finance dashboard
- `email.gs`: email templates and notifications
