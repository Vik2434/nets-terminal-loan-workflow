# Home

Welcome to the NETS Terminal Loan Workflow wiki.

This project is a Google Apps Script web application that uses Google Sheets as its data store and HtmlService pages for request intake, tracking, IT assignment, approval, finance visibility, and email notifications.

## Key Facts

- Runtime: Google Apps Script (V8)
- Data store: Google Sheets
- UI: HtmlService templates
- Access control: `User_Roles` sheet
- Notifications: MailApp
- Deployment style: raw Apps Script source with optional `clasp`
